# 2D Animated Short Film

## Project Overview

This project was created as a 2D animated short film. All characters, backgrounds, and visual assets were illustrated from scratch in Krita and animated in Adobe After Effects. The final scenes were edited and assembled in Adobe Premiere Pro.

The project includes character animation, camera movement, lighting effects, particle effects, and environmental storytelling.
